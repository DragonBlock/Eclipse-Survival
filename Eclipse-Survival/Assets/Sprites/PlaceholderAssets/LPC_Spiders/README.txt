Name:
  [LPC] Spider
Version:
  1
Description:
  This is a commissioned work by Redshrike. Included are 11 sprite sheets setup similar to the other LPC assets (up, left, bottm, right). All variations are of the same spider with different markings and color coats. There are 3 animations included, walking, attacking, and dying (only 1 direction dying).
